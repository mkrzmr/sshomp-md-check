SSHOMP Metadata Validator

Tools and Services PIDs only! Other item types will be added.
This tool checks the available metadata for an item in the SSHOMP against the list of recommended metadata and shows a table with the results.
This tool is meant for MP users who want to improve the quality of their entries, and MP curators who quickly want to check the metadata completeness of an item.
No data is logged or written back to the Marketplace. Users can export the results for convinience.
